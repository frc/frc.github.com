    Point of View

    Press ESC to enter the slide overview. Hold down alt and click on any element to zoom in on it using zoom.js. Alt + click anywhere to zoom back out.

    Perl

    PERL
    Use Image::Magick;

use warnings;
use strict;

my $dir = '/Users/Frantic/images';
my $pdffile = '/Users/Frantic/portfolio.pdf';

chdir( $dir ) or die "Cant chdir to $dir $!";
while( <*.png> ) {
  push @files, $_;
}

my $pdf = Image::Magick->new();

foreach $file ( sort @files ) {
  my $img = Image::Magick->new();
  $rc = $img->Read( $file );
  warn $rc if $rc;
  push @$pdf, $img;
}

$pdf->[0]->Coalesce();
$pdf->Write( $pdffile );

    Custom code

    Toggle code

    • Link to other slides using <a href="#/15">. The corresponding hash-tags can be found in the URL of the page you are linking to.

    • Add <section data-state="color"> to activate background change. List of available classes:

      green cyan orange red light-gray gray dark-gray

    • Add class="fragment" to elements to activate a step-based slide. It will display one fragment (block) after another. See example.
